== Description ==

WooTweaks Storefront Theme Tweak enhances the Storefront theme for WooCommerce with easy-to-use customization options directly in the WordPress admin. Perfect for developers and store owners, this plugin offers real-time toggles, CSS editing with modern CodeMirror integration, and seamless updates from GitHub, including private repositories with API token encryption.

Features include:

* Toggle Storefront style overrides on/off.
* Activate a custom middle menu with menu location support.
* Edit front-end and admin CSS with real-time syntax highlighting and color swatches.
* User-friendly admin interface with tabbed settings.
* Automatic updates via GitHub, supporting private repos with API tokens.

== Installation ==

1. Download `wt-storefront-theme-tweak.zip` from [releases](https://github.com/bodfather/wt-storefront-theme-tweak/releases).
2. Upload via **Plugins > Add New > Upload Plugin** in WordPress.
3. Activate the plugin from the **Plugins** menu.
4. Go to **WooTweaks > WT StoreFront Theme Tweak** under the admin menu to configure.

== Frequently Asked Questions ==

= How do I update the plugin? =
The plugin auto-checks for updates from GitHub. For private repos, save a GitHub API token in the settings.

= Can I edit CSS without coding knowledge? =
Yes, the CodeMirror editor highlights syntax and shows color swatches, making edits intuitive.

= How do I use a private GitHub repository? =
Set your repo to private, then add a Personal Access Token (repo scope) in the **API Token** tab.

= How do I add a middle menu? =
Enable the "Activate Middle Menu" toggle, then assign a menu in **Appearance > Menus > Manage Locations**.

= Where can I get support? =
Visit the [GitHub Issues page](https://github.com/bodfather/wt-storefront-theme-tweak/issues).
